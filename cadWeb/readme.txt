- Instalar o Xampp

- executar o servidor MySQL e Apache do xampp

- colocar a pasta cadWeb no diretório C: xampp > htdocs

- executar na porta padrão do xampp

- rodar o script ( localizado em script.txt ) MySQL pelo localhost/phpmyadmin 

- acessar o projeto por:  http://localhost/cadWeb
	
	- você será direcionado para a página de login do sistema;
	- cadastre um usuario para ter acesso ao sistema de cadastro de clientes;
	- caso executado corretamente, seu usuário aparecerá no painel do phpmyadmin.
